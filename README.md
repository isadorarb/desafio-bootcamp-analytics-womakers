# desafio-bootcamp-analytics-womakers
Código do desafio da Etapa 3 do Processo Seletivo do Bootcamp de Data Analytics da WoMakersCode

Para o Processo Seletivo do Bootcamp de Data Analytics da WoMakersCode, a primeira etapa foi a inscrição, a segunda etapa foi um curso de Python para iniciantes e a terceira etapa foi um desafio de código com a resolução de 5 exercícios propostos.

O curso de Python para iniciantes cobriu os seguintes tópicos:
- Conceitos básicos de Python;
- Tomada de Decisão e Laços de Repetição;
- Estrutura de Dados;
- Funções e Manipulação de Arquivos.
